# Olist Product Analytics Project

## Project Overview

This project analyses the Olist Brazilian E-Commerce dataset using SQL and Power BI to understand customer behaviour, revenue performance, order fulfilment, delivery delays, review scores, repeat purchasing, and retention patterns.

The goal is to simulate a real-world product analytics workflow used by product, business, and operations teams.

## Business Objectives

- Understand the customer journey from order creation to delivery.
- Analyse revenue performance across product categories, sellers, states, and payment methods.
- Identify delivery delays and operational bottlenecks.
- Evaluate the relationship between delivery performance and customer review scores.
- Measure repeat purchase and customer retention behaviour.
- Build an executive Power BI dashboard with actionable business recommendations.
